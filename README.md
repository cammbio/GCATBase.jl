# `GCATBase.jl`

This package is designed for analyzing DNA or RNA sequences based on coding theory and the structure of the genetic code. It serves as a base package and is supplemented by additional packages. Examples for coding theory in molecular biology are below.

This packages uses and compatible to [`BioSequences.jl`](https://github.com/BioJulia/BioSequences.jl).
